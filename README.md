# LangurLLM

## Installation

1. Clone the repository

```sh
git clone https://github.com/langur-treehacks/llm.git
```

2. Set up .env file (refer to .env.example):

```sh
touch .env
nano .env
```

3. Install the dependencies

```sh
pip install -r requirements.txt
```

5. Run the app

```sh
python ./app.py
```
